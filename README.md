# Proyecto de Formulario de preguntas 

Diseñar e implementar un formulario de encuesta interactivo que optimice la experiencia del usuario mediante validación en tiempo real y múltiples tipos de entrada, retroalimentación de las respuestas proporcionadas, proporcionado una herramienta util y de fácil manejo para la recolección de información en algun tema.

Desarrollado con:
React : Biblioteca de JavaScript de código abierto diseñada para construir interfaces de usuario (UI). Escogida por su rapidez y eficiencia engestión del estado con Hooks(useState, useEffect).
Next.js: Framework de React de código abierto desarrollado por Vercel que facilita la creación de aplicaciones web rápidas y con una configuraci´on minima.
TypeScript: TypeScript es un lenguaje de programación fuertemente
tipado que se basa en JavaScript, lo que le brinda mejores herramientas
a cualquier escala, mediante detección temprana de errores, autocompletado inteligente y refactorización segura.
TailwindCSS: Framework CSS utility-first que permite crear interfaces personalizadas directamente en JSX.Reduce significativamente el
tiempo de desarrollo y garantiza consistencia visual en toda la aplicaci´on
